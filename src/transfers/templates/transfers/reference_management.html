Инструкции по управлению справочниками
Статусы

Статусы определяют принадлежность записи (Бизнес, Личное, Налог и т.д.). Можно добавлять новые статусы по мере необходимости.

Типы операций

Типы операций определяют направление движения средств (Пополнение, Списание и т.д.). Список можно расширять новыми типами.

Категории

Основные категории для группировки расходов и доходов (Инфраструктура, Маркетинг и т.д.). Каждая категория может иметь несколько подкатегорий.

Подкатегории

Детализация внутри категорий (например, VPS, Proxy для категории Инфраструктура). Подкатегория обязательно должна быть связана с категорией.
